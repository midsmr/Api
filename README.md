## 概述
本程序基于Laravel构建

提供的集成的接口均为技术手段抓包或者第三方开源仓库

由开源作者：[@midsmr](https://github.com/midsmr) 提供永久更新维护

欢迎大家给仓库( [midsmr/Api](https://github.com/midsmr/Api) )点上星星

希望大家可以Fork本仓库一起让这个程序变得更强大

## 已集成接口

> 域名备案查询
> 
> Qr二维码生成
